# rust_WebDev

A minimal starter repo for building web apps in Rust. This README outlines goals, recommended tools, a quick setup, and common workflows so you can stop fighting JavaScript.

## Project goals

- Learn and use Rust for web development.
- Provide a small, reusable starter layout for APIs, server-rendered apps
